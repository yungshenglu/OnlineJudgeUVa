# UVA-10071 - Back to High School Physics

---

The input will contain two integers in each line. Each line makes one set of input. These two integers denote the value of $v$ ($âˆ’100 \le v \le 100$) and $t$ ($0 \le t \le 200$) ($t$ means at the time the particle gains that velocity)
